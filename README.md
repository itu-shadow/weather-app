# 🌦️ Weather App

A dynamic weather application that fetches real-time weather data and updates background images based on the searched city.

## Features
- Real-time weather data
- Dynamic background images per city
- Clean UI with glass effect

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
- Unsplash API

## How to Run
Open index.html in browser and search any city
